# 🧩 3D Maze Game

A simple 3D maze game built using Unity where the player navigates through a maze to find the exit.

---

## 🎮 How to Play

- Use **W / A / S / D** keys to move
- Navigate through the maze
- Find the exit to win

---

## 🛠 Built With

- Unity 2022.3 LTS
- C#
- Unity 3D Physics & Lighting

---

## 📦 Requirements

- Unity 2022.3.x LTS or later
- Windows / MacOS
- Basic knowledge of Unity (for modifications)
